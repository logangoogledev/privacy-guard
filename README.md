# Legal Policy Scanner

Chrome extension that scans websites for:

- Privacy Policies
- Terms of Service
- Legal Agreements
- Cookie Policies

And warns users about:

- Forced arbitration
- Data selling
- AI training clauses
- Third-party sharing
- Broad data collection

## Installation

1. Open Chrome
2. Go to:
   chrome://extensions/

3. Enable:
   Developer Mode

4. Click:
   Load unpacked

5. Select this folder

## Features

- Auto policy detection
- Risk scanner
- Severity scoring
- Popup results
